Este repositorio contiene la entrega final sobre el desarrollo del caso de uso expuesto a continuacion:

La universidad desea implementar una plataforma digital que permita centralizar el proceso de prácticas profesionales para sus programas de pregrado. Actualmente, este proceso se gestiona de forma desorganizada a través de correos, documentos físicos y formularios independientes, lo cual genera pérdida de información, demoras en las aprobaciones y falta de trazabilidad.
La solución debe permitir a los estudiantes postularse a ofertas, a las empresas publicar vacantes y registrar informes, y a la universidad realizar seguimiento de cada etapa del proceso.
